## API Used 

Spoonacular. (n.d.). Spoonacular Food API. Retrieved November 6, 2025, from https://spoonacular.com/food-api

This project uses the Spoonacular API
 (Spoonacular, n.d.) to fetch recipe data based on user inputs such as meal name, ingredients, and dietary restrictions.

Example request and response from the API:

Request:

GET https://api.spoonacular.com/recipes/complexSearch?query=pasta&maxFat=25&number=2

Response:
